# UI тестирование



# Адаптивность

- десктопная версия : ошибка "Epic sadface: Username and password do not match any user in this service" выходит за границы , верстка не ломается
- мобильная  версия : ошибка "Epic sadface: Username and password do not match any user in this service" выходит за границы



# Обязательные поля

- Поле «Username» не помечено символом *

- Поле «Password» не помечено символом *



# Валидация



# Тест кейс 1 : Ввод корректный данных

1. ввести валидный Login и password 
2. нажать кнопку "login"

Ожидаемый результат : пользователь успешно авторизуется



# Тест кейс 2 :пустые поля

1. Оставить поля пустые 
2. Нажать кнопку "login" 

Ожидаемый результат : отображается сообщение об ошибке 



# Тест кейс 3: невалидные данные 

1. ввести некоректные данные 
2. Нажать кнопку "login"

 Ожидаемый результат : появляется сообщение об ошибке 



# Проверка сетевых запросов


- При нажатии кнопки Login отправляется сетевой запрос

- Статус ответа: 200 

- Данные передаются корректно
